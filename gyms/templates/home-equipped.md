# Equipped Home Gym

Home gym with rack, barbell, and dumbbells.

## Typical Setup

### Essential
- Power rack or squat stand
- Barbell (Olympic)
- Weight plates (300+ lbs total)
- Flat/adjustable bench
- Pull-up bar (on rack)

### Common Additions
- Dumbbells (adjustable or fixed set)
- EZ curl bar
- Dip attachment
- Resistance bands
- Landmine attachment

### Nice to Have
- Cable pulley system
- Specialty bars (trap bar, SSB)
- GHD
- Rower or bike

## Full Exercise Library

### Barbell Movements
- Bench Press (flat, incline, close grip)
- Squat (back, front, pause)
- Deadlift (conventional, sumo, deficit)
- Overhead Press
- Barbell Row
- Romanian Deadlift
- Hip Thrust
- Good Morning

### Dumbbell Movements
- All pressing variations
- Rows (single arm, chest supported)
- Lunges, Bulgarian splits
- Curls, extensions
- Lateral raises

### Bodyweight
- Pull-ups, chin-ups, muscle-ups
- Dips
- Inverted rows
- Push-up variations
- Core work

### Cable/Pulley (if available)
- Lat pulldowns
- Cable rows
- Face pulls
- Tricep pushdowns
- Cable flyes

## Sample Programs

### Push Day
1. Bench Press: 4x5
2. Incline DB Press: 3x8
3. OHP: 3x6
4. Lateral Raises: 3x12
5. Tricep Pushdowns: 3x12

### Pull Day
1. Barbell Row: 4x6
2. Pull-ups: 4x max
3. Face Pulls: 3x15
4. DB Curls: 3x10
5. Hammer Curls: 2x12

### Leg Day
1. Squat: 4x5
2. Romanian Deadlift: 3x8
3. Bulgarian Split Squat: 3x8 each
4. Leg Curl (or Nordic): 3x10
5. Calf Raises: 4x15

### Full Body (3x/week option)
**Day A:**
1. Squat: 3x5
2. Bench Press: 3x5
3. Barbell Row: 3x5
4. Dips: 2x10
5. Curls: 2x12

**Day B:**
1. Deadlift: 3x5
2. OHP: 3x5
3. Pull-ups: 3x max
4. Lunges: 2x10 each
5. Face Pulls: 2x15

## Notes
- Full commercial gym capability
- No waiting for equipment
- Can train any time
- Loud music/chalk/no shirt = OK
- Easy to superset efficiently
