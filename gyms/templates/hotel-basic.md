# Basic Hotel Gym

Typical hotel fitness center with minimal equipment.

## Equipment

### Cardio
- Treadmill: 1-2
- Stationary bike: 1
- Elliptical: 0-1

### Strength
- Dumbbells: Up to 50 lbs (often 5-50 lb set)
- Adjustable bench: 1
- Cable machine: Usually none
- Resistance bands: Sometimes

### Other
- Exercise balls
- Yoga mats
- Mirror

## Workout Adaptations

### Upper Body Push
- DB Bench Press
- DB Shoulder Press
- Push-ups (various grips)
- DB Flyes
- Tricep kickbacks

### Upper Body Pull
- DB Rows
- Bent over DB flyes
- Curls
- If bar available: Chin-ups

### Lower Body
- Goblet squats
- DB Romanian deadlifts
- Lunges (all variations)
- Bulgarian split squats
- Single-leg deadlifts

### Core
- Planks
- Dead bugs
- Russian twists
- Mountain climbers

## Sample Hotel Workout (30-40 min)

### Full Body Option
1. Goblet Squat: 3x12
2. Push-ups: 3x15
3. DB Row: 3x10 each
4. Walking Lunges: 3x10 each
5. DB Shoulder Press: 3x10
6. Plank: 3x45s

### Upper Focus
1. DB Bench: 4x10
2. DB Row: 4x10 each
3. DB Shoulder Press: 3x12
4. DB Curls: 3x12
5. Tricep Kickbacks: 3x12
6. Push-ups: 2x max

## Notes
- Expect limited weight selection
- Focus on higher reps with available weights
- Supersets work well for efficiency
- Often 24-hour access
