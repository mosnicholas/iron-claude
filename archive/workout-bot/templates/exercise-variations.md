# 8-Week Exercise Rotation Reference

Keep exercise variety high without sacrificing progress. Rotate exercises every 2 weeks while maintaining the same movement patterns.

---

## How to Use This

1. Check which week you're on (1-8)
2. Find your exercises in the tables below
3. After Week 8, restart at Week 1
4. Track weights for each exercise variation

---

## Day 1: Upper Push (Monday)

| Week | Main Press | Secondary Press | Dip Variation | Isolation |
|------|-----------|-----------------|---------------|-----------|
| **1-2** | Barbell OHP | Incline DB Press | Weighted Dips | Cable Lateral Raise |
| **3-4** | DB Shoulder Press | Low Incline Barbell | Ring Dips | DB Lateral Raise |
| **5-6** | Push Press | Incline Smith | Bench Dips (weighted) | Machine Lateral |
| **7-8** | Landmine Press | Incline DB (neutral grip) | Parallel Bar Dips | Cable Y-Raise |

### Movement Patterns
- **Main Press**: Vertical push (shoulder dominant)
- **Secondary Press**: Incline push (upper chest + front delt)
- **Dip Variation**: Dip pattern (chest + tricep)
- **Isolation**: Lateral delt

---

## Day 2: Lower Strength (Tuesday)

| Week | Main Squat | Hinge | Single Leg | Hamstring |
|------|-----------|-------|------------|-----------|
| **1-2** | Back Squat | Romanian DL | Bulgarian Split Squat | Lying Leg Curl |
| **3-4** | Front Squat | Stiff-Leg DL | Reverse Lunge | Nordic Curl |
| **5-6** | Pause Squat | Good Mornings | Step-Ups (weighted) | Seated Leg Curl |
| **7-8** | Safety Bar Squat | Single-Leg RDL | Walking Lunge | Slider Leg Curl |

### Movement Patterns
- **Main Squat**: Bilateral squat (quad dominant)
- **Hinge**: Hip hinge (hamstring/glute)
- **Single Leg**: Unilateral leg (balance + strength)
- **Hamstring**: Knee flexion isolation

---

## Day 3: Conditioning (Wednesday)

| Week | Style | Description |
|------|-------|-------------|
| **1** | HIIT Intervals | 8-10 rounds: 30s all-out / 60s rest (bike/rower) |
| **2** | Circuit | KB swings, burpees, run, box jumps (4 rounds) |
| **3** | Cardio + Core | 20-25 min steady state + core work |
| **4** | EMOM | 20-min EMOM with varied movements |
| **5** | HIIT Intervals | Increase to 10-12 rounds |
| **6** | Circuit | Add 1 round or increase weight |
| **7** | Cardio + Core | Increase duration or add exercises |
| **8** | EMOM | Harder movements or more reps |

---

## Day 4: Upper Pull (Thursday)

| Week | Vertical Pull | Horizontal Row | Secondary Row | Rear Delt |
|------|--------------|----------------|---------------|-----------|
| **1-2** | Weighted Pull-ups | Barbell Row | Seated Cable Row | Face Pulls |
| **3-4** | Weighted Chin-ups | Pendlay Row | Chest-Supported DB Row | Reverse Pec Deck |
| **5-6** | Lat Pulldown (wide) | T-Bar Row | Single-Arm DB Row | Band Pull-Aparts |
| **7-8** | Neutral Grip Pull-up | Meadows Row | Machine Row | Cable Rear Delt Fly |

### Movement Patterns
- **Vertical Pull**: Lat dominant (pull down/up)
- **Horizontal Row**: Mid-back (rhomboids, lats, traps)
- **Secondary Row**: Additional horizontal pull volume
- **Rear Delt**: Posterior deltoid + external rotators

## Day 5: Full Body (Friday)

| Week | Hinge | Push | Pull | Single Leg |
|------|-------|------|------|------------|
| **1-2** | Trap Bar DL | DB Bench | Chin-ups | Walking Lunges |
| **3-4** | Conventional DL | Incline DB Press | Pull-ups | Goblet Squats |
| **5-6** | Sumo DL | Floor Press | Lat Pulldown | Rear-Foot Elevated Split |
| **7-8** | Deficit DL | DB Squeeze Press | Inverted Rows | Cossack Squats |

### Movement Patterns
- **Hinge**: Deadlift variation (posterior chain)
- **Push**: Horizontal press (chest dominant)
- **Pull**: Vertical pull (back)
- **Single Leg**: Unilateral lower body

## Exercise Substitutions

If equipment isn't available, use these swaps:

### Squat Variations
| If missing | Use instead |
|------------|-------------|
| Safety Bar | High Bar + Pad |
| Pause Squat | Tempo Squat (3-0-1) |

### Deadlift Variations
| If missing | Use instead |
|------------|-------------|
| Trap Bar | Conventional or Sumo |
| Deficit DL | Conventional from floor |

### Pull Variations
| If missing | Use instead |
|------------|-------------|
| Pendlay Row | Strict Barbell Row |
| Meadows Row | Single-Arm Landmine Row |
| T-Bar Row | Landmine Row |

### Press Variations
| If missing | Use instead |
|------------|-------------|
| Landmine Press | Single-Arm DB Press |
| Ring Dips | Weighted Parallel Dips |
| Floor Press | Close-Grip Bench |

---

## Customization Tips

This is a template - adjust based on your:
- **Equipment availability**: Swap exercises as needed
- **Training goals**: Add/remove movements
- **Recovery capacity**: Adjust volume (sets) per session
- **Schedule**: Reorganize days to fit your week
